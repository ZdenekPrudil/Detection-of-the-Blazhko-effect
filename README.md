# Detection of the Blazhko effect

Simple [Multi-layer Perceptron classifier](https://scikit-learn.org/stable/modules/generated/sklearn.neural_network.MLPClassifier.html "MLP sklearn") trained on the *I*-band photometric data from the OGLE-IV survey for the Galactic bulge ([1](https://ui.adsabs.harvard.edu/abs/2019AcA....69..321S/abstract "Cite"),[2](https://ui.adsabs.harvard.edu/abs/2014AcA....64..177S/abstract "Cite")) together with analysis of the Blazhko effect among fundamental mode RR Lyrae stars
